Смартфон Xiaomi Redmi Note 10S
Смартфон Xiaomi Redmi Note 10S с AMOLED-экраном диагональю 6,43 дюйма и разрешением 2400х1080. Процессор MediaTek Helio G95, видеоускоритель ARM Mali-G76 MC4. Емкость аккумулятора – 5000 мАч, зарядка 33 Вт. Основная камера – 64+8+2+2 МР, фронтальная – 13 МР.
В Xiaomi Redmi Note 10S установлена 6-43-дюймовая AMOLED-матрица разрешением 2400х1080 пикселей с пиковой яркостью 1100 нит. Поддержка цветовой гаммы DCI-P3, 8-битной цветности, режима чтения 3.0. Сертификация SGS свидетельствует о минимальном уровне синего излучения.
Разрешение основной камеры составляет 64 мегапикселя с диафрагмой f/1.79. За панорамную съемку отвечает 8-мегапиксельный датчик, за макросъемку – 2-мегапиксельный модуль. Есть датчик глубины разрешением 2 мегапикселя для портретной съемки. Поддерживается запись видео в разрешении 4К. Фронтальная 13-мегапиксельная камера с искусственным интеллектом записывает видео в 1080р.
Смартфон Redmi Note 10S оснащается процессором MediaTek Helio G95 с тактовой частотой 2,05 ГГц и графическим чипом ARM Mali-G76 MC4. Оперативная память представлена типом LPDDR4X, а встроенное хранилище – UFS 2.2.
Емкость аккумуляторной батареи равняется 5000 мАч. Для зарядки используется зарядный блок мощностью 33 Вт с кабелем USB Type-C. Корпус защищен от пыли и влаги по стандарту IP53.
Для бесконтактной оплаты задействуется модуль NFC. Беспроводные технологии представлены Wi-Fi 2,4 ГГц/5 ГГц и Bluetooth 5.0. Установлено 2 динамика, которые сертифицированы по стандарту Hi-Res. Есть 3,5 мм разъем для наушников.
